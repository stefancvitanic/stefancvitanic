149803 Maple – Soft Combination Stains Color Block Set

Our Auburn Combination Stain is a deep orange to red color that moderately blends the color of the wood yet allows some of the deep wood grain to show through.

Combination Stain

A Combination Stain uses both a Spray No Wipe Stain and a Wiping Stain to achieve a specific color.

Common Wood Species and Grade combinations when using this stain color:

  • Maple – Soft (Select/Premium Grade)
  • Maple – Soft (Natural/Standard Grade)
  • Maple – Soft – Curly (Natural/Standard Grade)

    Color Samples

    This is an In-Stock Color Block and is available for purchase as an Individual Color Block or as part of a Color Block Set.
